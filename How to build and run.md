How to build and run

This is a C++ console app I built in Visual Studio 2022. To run it yourself:


Clone or download this repo.
Open ProjectTwo.cpp in Visual Studio 2022, or add it to a new Empty Console App project.
Make sure the course data file is in the same folder as the program so it can find it when it loads.
Build the solution (Ctrl+Shift+B), then run it (Ctrl+F5).
Use the on-screen menu to load the data, print the full sorted course list, or look up a single course and its prerequisites.
